# PT. Afico Utama Group — Company Profile Website

Website company profile resmi untuk **PT. Afico Utama Group**, perusahaan jasa *Pest Control*, *Termite Control*, *Fumigation*, *Cleaning Service*, *General Cleaning*, *Manpower Supply*, dan *Landscape / Gardening* yang berbasis di Surabaya, Jawa Timur.

## Tentang

Website ini dibangun sebagai company profile satu halaman (single-page) yang menampilkan:

- Profil perusahaan, visi, dan core values
- 7 lini layanan utama (Pest Control, Termite Control, Fumigation, Cleaning Service, General Cleaning, Manpower Supply, Landscape/Gardening)
- Proses kerja (survei → treatment → monitoring → laporan)
- Daftar klien korporat (hospitality, kesehatan, industri, program MBG/SPPG)
- Form kontak dan informasi kantor (Head Office & Branch Office)
- Optimasi SEO (meta tags, Open Graph, structured data Schema.org, sitemap, robots.txt)

## Struktur Folder

```
.
├── index.html          # Halaman utama
├── style.css            # Seluruh styling (desain "Inspection Ticket")
├── script.js            # Interaktivitas (nav, scroll-reveal, filter klien, form)
├── robots.txt           # Konfigurasi crawler SEO
├── sitemap.xml           # Sitemap untuk mesin pencari
└── assets/
    └── logo-icon-transparent.png   # Logo perusahaan (background transparan)
```

## Teknologi

- **HTML5** — markup semantik dengan structured data (JSON-LD)
- **CSS3** — custom design system (CSS variables), responsive layout, tanpa framework
- **Vanilla JavaScript** — tanpa dependency/library eksternal

## Menjalankan Secara Lokal

Karena website ini murni HTML/CSS/JS statis, cukup buka `index.html` langsung di browser, atau jalankan local server sederhana:

```bash
python3 -m http.server 8080
```

Lalu buka `http://localhost:8080` di browser.
